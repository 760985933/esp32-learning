① 传感器

🌡️
心智图像：传感器是设备的「感官」。按接口分三类：模拟（电压连续变化→ADC 读）、数字单总线（一根线发脉冲）、总线型(I2C/SPI)（两根线挂很多设备）。认准接口，驱动就定了。

模拟量（电位器 / 光敏 / 可调电压）

import machine
adc = machine.ADC(machine.Pin(34)); adc.atten(machine.ADC.ATTN_11DB)
while True:
    print("原始值", adc.read())   # 0~4095

细节 + 接线图见 中级1 · ADC 模拟输入。

I2C 数字传感器（BME280 温湿度气压）

from machine import I2C, Pin
i2c = I2C(0, scl=Pin(22), sda=Pin(21))
# 用 bme280 库: from bme280 import BME280; b=BME280(i2c=i2c); print(b.values)

I2C 地址点名机制 + 接线见 中级5 · I2C 与 OLED。

② 电机（让设备动起来）

⚙️
心智图像：GPIO「力气小」，必须雇「壮汉」驱动板。PWM 调速度，方向由逻辑脚决定。舵机看脉宽定角度，步进看脉冲数定步数。
from machine import PWM, Pin
motor = PWM(Pin(14), freq=1000)   # → 驱动板 IN1
motor.duty_u16(32768)              # 50% 半速
# 舵机: from machine import Servo; Servo(Pin(13)).angle(90)
铁律绝不用 GPIO 直接驱电机！感性负载反向电压会击穿芯片；驱动板要接独立电源并与 ESP32 共 GND；务必加续流二极管。接线图见 高级5。

③ 屏幕

OLED（I2C，最常用）

from machine import I2C, Pin
import ssd1306
i2c = I2C(0, scl=Pin(22), sda=Pin(21))
oled = ssd1306.SSD1306_I2C(128, 64, i2c)
oled.text("Hello ESP32", 0, 0); oled.show()

接线图（VCC/GND/SDA/SCL）见 中级5 · I2C 与 OLED。

I2C vs SPI 屏幕OLED 小屏用 I2C（省脚、慢）；大彩屏(TFT)用 SPI（脚多、快）。选屏先看接口，再找对应库。
